一、总体玩法概述
（下载方式见最下面）
通过极高难度的锻造过程，添加新的极品装备自定义打造
本插件构建“管理员配置-玩家实操”的双端协同锻造体系，核心目标是打破原版合成的单一性，为玩家提供沉浸式、个性化的武器打造体验。
核心逻辑：管理员通过配置属性、模板、材料关系及武器配方，搭建基础锻造规则框架；玩家通过学习模板、摆放材料、互动锻造获取武器部件，最终合成成品武器。整个流程融合“策略配置”与“操作技巧”，既支持管理员按需定制玩法难度与多样性，也让玩家在材料搭配、锻造操作中获得探索乐趣。
核心特色：
•高度自定义：属性、模板、材料关系、武器配方全维度可配置，适配不同服务器玩法需求；
•沉浸式操作：锻造阶段引入动态互动机制，需玩家精准操作把控成功率，打破传统合成的被动等待模式；
•多插件兼容：支持AP3.0、SX、原版及APS等常见属性插件的属性映射，无需修改现有服务器体系，集成成本低；
•分层玩法：从材料锻造到武器合成，流程分层清晰，逐步提升难度，兼顾新手友好度与高阶玩家探索性。
宣传标语：全维度自定义配置，打造独一无二的锻造规则体系
二、完整流程演示
本流程覆盖“管理员配置-玩家锻造-武器合成”的全链路，清晰呈现各环节的衔接逻辑与核心操作要点。
（一）前置配置阶段（管理员）
1.创建属性：管理员手持属性对应材质，输入/dt at create 属性名称，完成属性创建与材质绑定；
2.建立属性映射：输入/dt at ys 本插件属性 服务器属性插件属性 百分比，实现与现有属性插件的兼容；
3.创建武器部件模板：手持模板材质（空手默认纸），输入/dt mb create 模板名称进入编辑页面，点击淡灰色玻璃板确定必要槽位，通过紫水晶设置武器部件材质，红石块保存退出；
4.编辑材料：手持材料输入/dt cl bj进入属性编辑页面，修改熔点、复杂度等属性；通过/dt pc 材料1 材料2 排斥值和/dt gm 材料1 材料2 共鸣值设定材料互动关系；
5.创建武器配方：快捷栏放置武器部件模板，输入/dt wq create武器名称进入配方页面，配置武器部件模板及属性生效百分比，添加武器材质并保存。
（二）玩家锻造阶段
6.学习模板：使用指定道具（控制台执行/dt mb st 玩家名称 模板名称指令），学习管理员创建的武器部件模板；
7.选择模板：输入/dt进入模板选择页面，点击目标模板进入锻造页面；
8.材料摆放：在锻造页面将材料摆放到指定区域，覆盖所有必要槽位后，点击快捷栏铁砧开始锻造；
9.互动锻造：绿色玻璃板运动，进入材料格子时，玩家需梳理材料纹路并在最佳时机确认固化，获取完美/正确固化的材料（一般固化无效）；
10.合成武器：收集所有必要武器部件并放入快捷栏，输入/dt hc进入武器材质选择页面，点击确认后合成成品武器，属性自动叠加计算。
（三）核心规则衔接说明
•锻造成功率：由所有材料与四周8个格子内材料的排斥值总和决定，排斥值越高，成功率越低。计算逻辑为“基础成功率100% - 所有材料间排斥值总和×1%”，若排斥值总和为负数，则基础成功率会相应提升（例如排斥值总和为-5，成功率则为105%，实际按100%计算）；
•材料属性计算：固化后的材料属性先计算基础加成，公式为“材料基础属性×（1 +（成功梳理材料纹路数量 - 错误材料纹路数量）×10%）×（1.2 - 20%×固化指针与最佳固化点相距格数）”；再叠加共鸣值加成——每个材料与四周8个材料的共鸣值总和，1点共鸣值对应属性最终判定±1%，最终汇总为该武器部件的属性；
•成品属性：由各武器部件的属性生效值相加得出，属性映射自动适配服务器现有属性体系。成品的熔点为所有材料总和，复杂度为2*所有材料复杂度平均值
宣传标语：可视化GUI操作，轻松上手，精准把控每一步锻造细节

三、各GUI详细介绍
本插件所有操作均基于“大箱子”核心界面衍生，不同GUI承担不同功能，界面布局、操作逻辑及核心作用如下：
1. 模板编辑页面
触发方式：管理员输入/dt mb create 模板名称或/dt mb bj 模板名称指令。
界面布局：核心为大箱子界面，快捷栏包含“武器部件”紫水晶（第一格）、红石块（保存退出）。
核心功能：
•必要槽位配置：点击大箱子内的淡灰色玻璃板，标记该模板的必要槽位（玩家锻造需完全覆盖）；
•武器部件材质设置：通过快捷栏紫水晶返回背包，将武器部件放入副手后用/qd指令保存；
•保存退出：点击快捷栏红石块，自动保存当前配置并返回游戏主界面。
操作注意：未设置必要槽位或未配置武器部件材质时，保存会失败，需补全配置后重新操作。
2. 材料属性编辑页面
触发方式：管理员手持材料输入/dt cl bj指令。
界面布局：大箱子界面，第一排最中间展示当前编辑的材料材质，中间区域展示各属性对应的绑定材质（创建属性时关联的材质），快捷栏包含红石块（保存退出）。
核心功能：
•属性可视化展示：通过材质图标清晰区分不同属性，便于管理员快速定位目标属性；
•属性值修改：点击属性对应的材质图标，跳转至聊天框输入数值，回车即可保存并返回页面；
•必要属性默认展示：熔点、复杂度两个必要属性固定展示，无需额外创建，直接修改即可。其中，熔点默认值80，核心作用是影响互动锻造阶段绿色玻璃板的移动速度——绿色玻璃板在空白格子移动速度为1格/80tick，经过材料格子时速度变为1格/（材料熔点）tick，熔点越高，玻璃板停留时间越长，玩家操作窗口越充裕；复杂度默认值0，作用是决定互动锻造时物品栏亮起的“未梳理材料纹路”紫色玻璃板数量，复杂度越高，亮起的纹路数量越多，梳理难度越大，对玩家操作精准度要求更高，复杂度主要影响材料的最终上限。
操作注意：属性值需输入合法数字，否则保存无效，需重新输入。
3. 武器配方页面
触发方式：管理员快捷栏放置武器部件模板，输入/dt wq create 武器名称指令。
界面布局：大箱子界面，第2行展示武器部件模板，第4-5行展示武器材质，第6行包含“上一页”“下一页”烟花火箭、“添加材质”末影珍珠，快捷栏包含红石块（保存退出）。
核心功能：
•武器部件模板管理：右键第2行的模板可删除，左键可修改属性生效百分比；
•武器材质管理：点击第4-5行的材质可删除，点击“添加材质”末影珍珠返回背包，快捷栏放置材质后用/qd指令保存；
•分页浏览：通过第6行的烟花火箭切换武器材质页面，适配多材质配置需求。
操作注意：武器名称不可重复，否则无法创建新配方；至少配置1个武器部件模板和1种武器材质，否则保存失败。
4. 模板选择页面
触发方式：玩家输入/dt指令。
界面布局：大箱子界面，格子内展示玩家已学习的所有武器部件模板（按模板名称分类排列）。
核心功能：
•模板快速筛选：仅展示已学习的模板，避免无关模板干扰；
•锻造流程衔接：点击目标模板后，直接跳转至对应的材料摆放锻造页面，简化操作步骤。
操作注意：未学习任何模板时，页面无可用选项，需先通过道具学习模板。
5. 锻造页面（材料摆放+互动锻造）
触发方式：玩家在模板选择页面点击目标模板。
界面布局：
•材料摆放阶段：大箱子界面为材料摆放区域（必要槽位有特殊标识），物品栏为玩家物品栏，快捷栏包含铁砧（开始锻造）；
•互动锻造阶段：材料摆放区域不变，新增绿色玻璃板（动态运动）、快捷栏“固化指针”箭（进度条）和“最佳固化点”的钻石块，物品栏不再展示背包物品，改为展示随机出现的“未梳理材料纹路”紫色玻璃板（不会与确认固化按键重合）、“空白”淡灰色玻璃板、背包第27个展示“确认固化”铁砧。
核心功能：
•材料摆放引导：必要槽位的特殊标识帮助玩家快速定位，避免遗漏；
•动态锻造互动：绿色玻璃板按S形运动，直观展示锻造进度；固化指针和纹路梳理机制，提升操作趣味性与策略性；
•固化结果可视化：完美/正确/一般固化分别对应金色、绿色、黑色玻璃板，清晰反馈操作效果。其中，完美固化（相距0格）20%属性加成，正确固化（相距1~2格）有轻微属性折扣，一般固化（相距3格及以上）不计入属性统计，且必要槽位若出现一般固化则锻造直接失败；
操作注意：材料必须覆盖所有必要槽位才能点击铁砧开始锻造；互动锻造阶段需在绿色玻璃板停留材料格子期间完成纹路梳理与确认固化，否则该材料固化失败。
6. 武器材质选择页面
触发方式：玩家收集所有武器部件后，输入/dt hc指令。
界面布局：大箱子界面，格子内展示玩家已学习的所有武器材质（管理员通过/dt wq cz 玩家名称 武器材质名称指令赋予学习权限）。
核心功能：
•武器材质可视化选择：清晰展示所有可用武器样式，便于玩家自主选择；
•成品属性自动计算：点击确认后，系统自动汇总各武器部件的属性生效值，叠加生成最终武器属性，并适配服务器属性体系。
操作注意：需先收集对应武器配方要求的所有武器部件，否则无法进入该页面；未学习任何武器材质时，需联系管理员通过指令赋予权限。
宣传标语：指令全汇总，一键查阅，锻造操作零障碍
